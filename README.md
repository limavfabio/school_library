### Prerequisites

In order to run this project you need:

```sh
 gem install ruby
```

### Setup

Clone this repository to your desired folder:

```sh
  cd my-folder
  git clone https://github.com/limavfabio/school_library.git
```

### Run tests

To run tests, run the following command:

```sh
  rubocop .
```
